Concept

How it works

About Us

IOTemp helps you measure the temperature,humidity of a device , an open space or a closed space using a sensor.

IOTemp uses cloud for the sensor's connection , thanks to an ISP.

IOTemp provides an API (RESTful API) to acess and manage your data on a remote platform with simple and standard Internet tools.

Rest API

You need a token to be authenticated at IOTemp. Copy this token and paste it in your remote devices' scripts.

application, board, web.. They all can connect to IOTemp with REST API.
